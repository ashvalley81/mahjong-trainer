# 麻雀 点数計算トレーナー 🀄

符・翻・点数・役を実戦形式で練習できる、単一HTMLの麻雀点数計算トレーナーです。

**公開URL: https://ashvalley81.github.io/mahjong-trainer/**

## 練習モード

- **手牌から全部** — 牌姿を見て役・翻・符・点数をすべて自分で計算
- **符計算** — 手牌の構成（面子・待ち・ツモ/ロン）から符を計算
- **役の判定** — 成立している役を複数選択で当てる
- **符・翻→点数** — 条件から最終点数を計算（早見表の暗記）
- **早見表** — 子/親 × ロン/ツモの点数表

## 特徴

- 実際の牌姿を表示し、和了牌をハイライト
- **ドラ表示牌から自分でドラを数える**（表示牌の次の牌がドラ）
- 答え合わせ時に符の内訳・役の内訳を表示
- 標準リーチ麻雀ルール（20符ベース・平和20/30符・喰い平和30符・満貫以上の打ち切り）

## 使い方

ブラウザで `index.html` を開くだけ。インストール不要。
